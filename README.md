# BACKEND
## Установка:

Используется node.js, поэтому необходимо его установить.
https://nodejs.org/en

В командной строке или терминале:
Переходим в папку backend
```
npm install
```
## Запуск
<b>Windows / Linux</b>
В командной строке или терминале:
```
npm start / npm run dev
```
Потом заходим на сайт:
```
http://localhost:3000
```
